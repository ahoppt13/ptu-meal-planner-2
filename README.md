# PT:U Meal Planner

Personalised, evidence-based meal planning calculator for [PT:U Personal Training](https://www.pt-u.co.uk).

## Features

- Calorie calculator (Mifflin-St Jeor equation)
- Protein targeting (1.6-2.0g/kg based on ISSN guidelines)
- 5-day meal plans with full recipes
- Diet options: No Restrictions, Vegetarian, Vegan, Keto
- Health conditions: High Blood Pressure, High Cholesterol, Type 2 Diabetes
- Allergen filtering (8 common allergens)
- Scalable shopping list (1-10 people)
- Print/PDF export
- Lead capture (name, email, goals)

## Evidence Base

- DASH Diet for hypertension (NCBI PMC10551663)
- Low-GI diets for Type 2 Diabetes (PubMed 31374573)
- Cholesterol management via soluble fibre (PubMed 25430608)

